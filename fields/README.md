# Fields

## Entities and bundles

* Content types **SHOULD** use only singular name:
  * Correct:`standard_page`
  * Incorrect: `standard_pages`
* Vocabularies **SHOULD** use only plural name:
  * * Correct: ``authors or `tags```
    * Incorrect: `author` or `tag`

## Fields

* Fields **SHOULD** have entity name in the name, separated by

  underscore `_`: `field_author_name`, `field_author_mail`, `field_author_address`.
